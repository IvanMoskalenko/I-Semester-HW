# ProgrammingCourse

Доступные аргументы:

--first-task <x> - первая задача

--second-task <x> - вторая задача

--third-task - третья задача

--forth-task - четвёртая задача

--fifth-task - пятая задача

--sixth-task - шестая задача

--help - выводит список всех аргументов
